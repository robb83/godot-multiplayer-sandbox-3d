# Godot Multiplayer Sandbox 3D

## About the Project

The goal of this project is to learn and explore the capabilities of the Godot Engine. It serves as an experiment to see what everyday people can achieve using Godot, especially in the context of multiplayer 3D environments.

This project is not intended to be a polished product, but rather a sandbox for testing ideas, learning by doing, and pushing the boundaries of what can be built with limited experience.
